## ✨ Projeto

Nu Kenzie é um sistema de controle financeiro, o usuário pode cadastrar entradas e saídas e depois conferir se o saldo ficou positivo ou negativo, também há a opção de checar apenas saídas ou entradas separadamente.

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias/bibliotecas:

<table border="0">
 <tr>

<td> CSS3</td>
<td> JavaScript</td>
<td> React.JS</td>
<td> React Numeral</td>

<td> React Icons</td>
 </tr>

## 🌟 Features

-   [ ] Cadastro de finanças em entradas/saídas junto a uma descrição.
-   [ ] 100% responsivo em resoluções até 4k.
-   [ ] Filtro de despesas para consultas de modo individual.
-   [ ] Valor total de modo dinâmico, a partir do que está sendo mostrado em tela.

## 👨🏻‍💻 Executando o projeto

Utilize o **yarn** ou o **npm install** para instalar as dependências do projeto.
Em seguida, inicie o projeto.

Iniciando a versão web:

```cl
yarn start ou npm run dev
```
